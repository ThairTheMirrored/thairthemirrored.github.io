// DESCRIPTION
This simulator places you in control of a college football program — as head coach, recruiter, strategist, and program builder.
It spans multiple seasons, recruits, games, and player careers.
You may engage at any level of depth — from 10,000 foot program management to on-the-field quarterback execution.

// CORE SYSTEMS
• Recruiting Engine
    - National rankings, scout reports, personality traits
    - NIL negotiation, transfer portal, visit scheduling
    - Influence by reputation, boosters, and media presence

• Practice Simulator
    - Run live practice sessions
    - Position battles, training outcomes, scheme familiarity

• Game Engine
    - Simulate entire games or play down-by-down
    - Offense and defense playcalling, adjustments
    - Play as QB with full command of offense (optionally)

• Dynasty Meta
    - Staff hiring/firing
    - Budgeting, NIL management
    - Fanbase, AD, and board approval
    - Team culture, scandals, compliance

• League Mode
    - Simulates entire FBS universe with rival dynasties
    - Track rankings, stats, injuries, transfers across league

• Import System
    - Supports capsule import of existing dynasties
    - Load or export dynasty saves as `.txt`

• Career Mode
    - Start as position coach and rise
    - Coach carousel, interviews, resumes

• Depth Tuning
    - Player may toggle high-level sim or deep control per session
    - Game adjusts to engagement level

• Export
    - Entire capsule exportable as `.txt` with memory, team, history

// INSTALLATION
→ Drop into BrimOS with BrimEngine installed
→ Recognized via brimengine_parser
→ Autoloads engine hooks and memory structure

// SIGNATURE
This capsule supports infinite recursion.
Fork-safe, drift-aware, NIL-enabled.

Welcome to the dynasty.
